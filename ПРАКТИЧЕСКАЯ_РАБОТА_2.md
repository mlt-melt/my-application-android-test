# ПРАКТИЧЕСКАЯ РАБОТА Часть 2: Контейнеры для Размещения (Layout Managers)

## Цель

Освоить основные контейнеры размещения в Android:
- **LinearLayout** - линейное расположение элементов
- **RelativeLayout** - позиционирование относительно других элементов
- **TableLayout** - табличное расположение в строках и столбцах
- **FrameLayout** - слоистое расположение элементов
- **GridLayout** - расположение в сетке с гибким управлением
- **ScrollView / HorizontalScrollView** - прокручиваемое содержимое

Каждый контейнер демонстрируется в двух вариантах:
1. **XML декларативный** способ (в файлах layout)
2. **Java программный** способ (создание в коде)

---

## 1. LinearLayout: Линейное расположение

### Описание
`LinearLayout` проводит дочерние элементы **в одной линии** - либо горизонтально, либо вертикально.

### Ключевые атрибуты
- **`android:orientation`** - `vertical` или `horizontal`
- **`android:layout_weight`** - распределение оставшегося пространства (требует width/height = 0)
- **`android:gravity`** - выравнивание содержимого контейнера
- **`android:layout_gravity`** - выравнивание элемента внутри контейнера

### Пример 1: Равномерное распределение (Weight)

#### XML (layout_linear_weight.xml)
```xml
<LinearLayout
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:orientation="horizontal">
    
    <TextView
        android:layout_width="0dp"
        android:layout_height="50dp"
        android:layout_weight="1"
        android:background="#2196F3"
        android:text="Вес 1" />
    
    <TextView
        android:layout_width="0dp"
        android:layout_height="50dp"
        android:layout_weight="1"
        android:background="#4CAF50"
        android:text="Вес 1" />
    
    <TextView
        android:layout_width="0dp"
        android:layout_height="50dp"
        android:layout_weight="1"
        android:background="#FF6F00"
        android:text="Вес 1" />
</LinearLayout>
```

#### Java код (ProgrammaticLinearLayoutActivity.java)
```java
LinearLayout linearLayout = new LinearLayout(this);
linearLayout.setOrientation(LinearLayout.HORIZONTAL);

for (int i = 0; i < 3; i++) {
    TextView view = new TextView(this);
    view.setText("Вес 1");
    view.setTextColor(Color.WHITE);
    view.setGravity(Gravity.CENTER);
    
    LinearLayout.LayoutParams params = new LinearLayout.LayoutParams(
        0,  // width = 0 (распределяется по weight)
        50,
        1   // weight = 1
    );
    view.setLayoutParams(params);
    linearLayout.addView(view);
}
```

### Пример 2: Разные веса (1, 2, 3)
```java
// Элемент 1: вес 1 (1/6 ширины)
params = new LinearLayout.LayoutParams(0, 50, 1);

// Элемент 2: вес 2 (2/6 ширины)
params = new LinearLayout.LayoutParams(0, 50, 2);

// Элемент 3: вес 3 (3/6 ширины)
params = new LinearLayout.LayoutParams(0, 50, 3);
```

### Пример 3: layout_gravity (выравнивание в родителе)
```xml
<LinearLayout android:orientation="horizontal">
    <TextView
        android:layout_gravity="top"
        android:text="Вверху" />
    
    <TextView
        android:layout_gravity="center_vertical"
        android:text="По центру" />
    
    <TextView
        android:layout_gravity="bottom"
        android:text="Внизу" />
</LinearLayout>
```

## 2. RelativeLayout: Позиционирование относительно

### Описание
В `RelativeLayout` элементы позиционируются **относительно друг друга** или границ контейнера.

### Ключевые атрибуты (layout_* префикс)
- **`layout_alignParent*`** - выравнивание по границам родителя
  - `android:layout_alignParentTop="true"` - приклеить к верхней границе
  - `android:layout_alignParentBottom="true"` - приклеить к нижней границе
  - `android:layout_alignParentStart="true"` - приклеить к левой границе
  - `android:layout_alignParentEnd="true"` - приклеить к правой границе

- **`layout_*Of="@id/otherView"`** - позиционирование относительно другого элемента
  - `android:layout_rightOf="@id/button1"` - располагать справа от button1
  - `android:layout_below="@id/button1"` - располагать ниже button1
  - `android:layout_above="@id/button1"` - располагать выше button1
  - `android:layout_toStartOf="@id/button1"` - располагать слева от button1

- **`layout_align*`** - выравнивание границ с другим элементом
  - `android:layout_alignTop="@id/button1"` - верхняя граница совпадает с button1

### Пример: Размещение относительно элементов

#### XML (layout_relative.xml)
```xml
<RelativeLayout android:layout_width="match_parent" android:layout_height="200dp">
    
    <TextView
        android:id="@+id/title"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout_alignParentTop="true"
        android:layout_alignParentStart="true"
        android:text="Заголовок" />
    
    <TextView
        android:id="@+id/left_button"
        android:layout_width="80dp"
        android:layout_height="50dp"
        android:layout_below="@id/title"
        android:layout_alignParentStart="true"
        android:text="Левая" />
    
    <TextView
        android:id="@+id/center_view"
        android:layout_width="80dp"
        android:layout_height="50dp"
        android:layout_below="@id/title"
        android:layout_centerHorizontal="true"
        android:text="Центр" />
    
    <TextView
        android:layout_width="80dp"
        android:layout_height="50dp"
        android:layout_below="@id/title"
        android:layout_alignParentEnd="true"
        android:text="Правая" />
</RelativeLayout>
```

#### Java код (ProgrammaticRelativeLayoutActivity.java)
```java
RelativeLayout relativeLayout = new RelativeLayout(this);

TextView titleView = new TextView(this);
titleView.setId(View.generateViewId());
titleView.setText("Заголовок");

RelativeLayout.LayoutParams titleParams = new RelativeLayout.LayoutParams(
    RelativeLayout.LayoutParams.WRAP_CONTENT,
    RelativeLayout.LayoutParams.WRAP_CONTENT
);
titleParams.addRule(RelativeLayout.ALIGN_PARENT_TOP);
titleParams.addRule(RelativeLayout.ALIGN_PARENT_START);
titleView.setLayoutParams(titleParams);
relativeLayout.addView(titleView);

// Левый элемент
TextView leftView = new TextView(this);
leftView.setId(View.generateViewId());
leftView.setText("Левая");

RelativeLayout.LayoutParams leftParams = new RelativeLayout.LayoutParams(
    80, 50
);
leftParams.addRule(RelativeLayout.BELOW, titleView.getId());
leftParams.addRule(RelativeLayout.ALIGN_PARENT_START);
leftView.setLayoutParams(leftParams);
relativeLayout.addView(leftView);
```

## 3. TableLayout: Табличное расположение

### Описание
`TableLayout` - вертикальный LinearLayout с рядами (`TableRow`), где каждый ряд может содержать несколько столбцов.

### Ключевые атрибуты
- **`android:stretchColumns`** - индексы столбцов для растяжения
  - `android:stretchColumns="*"` - все столбцы распределяются равномерно
  - `android:stretchColumns="0,2"` - основные столбцы 0 и 2

- **`android:layout_span`** (на элементе TableRow) - объединение ячеек
  - `android:layout_span="2"` - элемент занимает 2 столбца

### Пример: Таблица с данными

#### XML (layout_table.xml)
```xml
<TableLayout
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:stretchColumns="*">
    
    <!-- Заголовок -->
    <TableRow android:background="#4CAF50">
        <TextView
            android:text="ID"
            android:textColor="@android:color/white"
            android:gravity="center"
            android:padding="8dp" />
        <TextView
            android:text="Название"
            android:textColor="@android:color/white"
            android:gravity="center"
            android:padding="8dp" />
        <TextView
            android:text="Цена"
            android:textColor="@android:color/white"
            android:gravity="center"
            android:padding="8dp" />
    </TableRow>
    
    <!-- Строка 1 -->
    <TableRow android:background="#F5F5F5">
        <TextView android:text="1" android:padding="8dp" />
        <TextView android:text="Товар A" android:padding="8dp" />
        <TextView android:text="100 ₽" android:padding="8dp" />
    </TableRow>
    
    <!-- Строка 2 -->
    <TableRow>
        <TextView android:text="2" android:padding="8dp" />
        <TextView android:text="Товар B" android:padding="8dp" />
        <TextView android:text="200 ₽" android:padding="8dp" />
    </TableRow>
</TableLayout>
```

#### Java код (ProgrammaticTableLayoutActivity.java)
```java
TableLayout table = new TableLayout(this);
table.setStretchAllColumns(true);

// Заголовок
TableRow headerRow = new TableRow(this);
headerRow.setBackgroundColor(0xFF4CAF50);

String[] headers = {"ID", "Название", "Цена"};
for (String header : headers) {
    TextView cell = new TextView(this);
    cell.setText(header);
    cell.setTextColor(Color.WHITE);
    cell.setGravity(Gravity.CENTER);
    cell.setPadding(8, 8, 8, 8);
    headerRow.addView(cell);
}
table.addView(headerRow);

// Строки данных
String[][] data = {
    {"1", "Товар A", "100 ₽"},
    {"2", "Товар B", "200 ₽"}
};
for (int i = 0; i < data.length; i++) {
    TableRow dataRow = new TableRow(this);
    if (i % 2 == 0) dataRow.setBackgroundColor(0xFFF5F5F5);
    
    for (String value : data[i]) {
        TextView cell = new TextView(this);
        cell.setText(value);
        cell.setPadding(8, 8, 8, 8);
        dataRow.addView(cell);
    }
    table.addView(dataRow);
}
```

### Пример: Слияние ячеек (columnSpan)
```xml
<TableRow>
    <TextView
        android:layout_span="2"
        android:text="Объединено 2 столбца" />
    <TextView android:text="Обычная" />
</TableRow>
```

## 4. FrameLayout: Слоистое расположение

### Описание
`FrameLayout` - самый простой контейнер. Элементы **накладываются друг на друга** (Z-order).
Последний добавленный элемент видна сверху.

### Ключевой атрибут
- **`android:layout_gravity`** - позиционирование элемента в FrameLayout
  - Поддерживает все комбинации: `top|start`, `center`, `bottom|end`, и т.д.

### Пример: Слои с позиционированием

#### XML (layout_frame.xml)
```xml
<FrameLayout
    android:layout_width="match_parent"
    android:layout_height="150dp"
    android:background="#E8F5E9">
    
    <!-- Фоновый слой -->
    <TextView
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:background="#4CAF50"
        android:text="Фон"
        android:gravity="center"
        android:textColor="@android:color/white" />
    
    <!-- Верхний левый слой -->
    <TextView
        android:layout_width="80dp"
        android:layout_height="50dp"
        android:background="#2196F3"
        android:text="Top-Left"
        android:gravity="center"
        android:textColor="@android:color/white"
        android:layout_gravity="top|start"
        android:layout_marginStart="8dp"
        android:layout_marginTop="8dp" />
    
    <!-- Центральный слой -->
    <TextView
        android:layout_width="80dp"
        android:layout_height="50dp"
        android:background="#FF6F00"
        android:text="Center"
        android:gravity="center"
        android:textColor="@android:color/white"
        android:layout_gravity="center" />
    
    <!-- Нижний правый слой (видна сверху) -->
    <TextView
        android:layout_width="80dp"
        android:layout_height="50dp"
        android:background="#D32F2F"
        android:text="Bottom-Right"
        android:gravity="center"
        android:textColor="@android:color/white"
        android:layout_gravity="bottom|end"
        android:layout_marginEnd="8dp"
        android:layout_marginBottom="8dp" />
</FrameLayout>
```

#### Java код (ProgrammaticFrameLayoutActivity.java)
```java
FrameLayout frameLayout = new FrameLayout(this);
frameLayout.setLayoutParams(new LinearLayout.LayoutParams(
    LinearLayout.LayoutParams.MATCH_PARENT, 150
));

// Фоновый слой
TextView bgLayer = new TextView(this);
bgLayer.setText("Фон");
bgLayer.setBackgroundColor(0xFF4CAF50);
bgLayer.setGravity(Gravity.CENTER);
FrameLayout.LayoutParams bgParams = new FrameLayout.LayoutParams(
    FrameLayout.LayoutParams.MATCH_PARENT,
    FrameLayout.LayoutParams.MATCH_PARENT
);
bgLayer.setLayoutParams(bgParams);
frameLayout.addView(bgLayer);

// Верхний левый слой
TextView topLeftLayer = new TextView(this);
topLeftLayer.setText("Top-Left");
topLeftLayer.setBackgroundColor(0xFF2196F3);
topLeftLayer.setTextColor(Color.WHITE);
topLeftLayer.setGravity(Gravity.CENTER);

FrameLayout.LayoutParams topLeftParams = new FrameLayout.LayoutParams(
    80, 50,
    Gravity.TOP | Gravity.START
);
topLeftParams.setMargins(8, 8, 0, 0);
topLeftLayer.setLayoutParams(topLeftParams);
frameLayout.addView(topLeftLayer);

// ... аналогично для других слоев
```

## 5. GridLayout: Сетка с гибким управлением

### Описание
`GridLayout` (из androidx.gridlayout) позволяет распределять элементы в **сетку** с контролем строк, столбцов, spans и веса.

### Ключевые атрибуты
- **`android:rowCount`** - количество строк
- **`android:columnCount`** - количество столбцов
- **`app:layout_row`** - номер строки элемента
- **`app:layout_column`** - номер столбца элемента
- **`app:layout_rowSpan`** - объединение строк
- **`app:layout_columnSpan`** - объединение столбцов
- **`app:layout_columnWeight`** - вес для распределения ширины (требует width=0)

### Пример: Сетка 3x3

#### XML (layout_grid.xml)
```xml
<GridLayout
    xmlns:app="http://schemas.android.com/apk/res-auto"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:rowCount="3"
    android:columnCount="3"
    android:background="@android:color/white">
    
    <!-- Заголовок строки -->
    <TextView
        android:text="Колонна 1"
        android:background="#1976D2"
        android:textColor="@android:color/white"
        android:gravity="center"
        android:padding="8dp"
        app:layout_row="0"
        app:layout_column="0"
        app:layout_columnWeight="1"
        android:layout_width="0dp"
        android:layout_height="50dp" />
    
    <TextView
        android:text="Колонна 2"
        android:background="#1976D2"
        android:textColor="@android:color/white"
        android:gravity="center"
        android:padding="8dp"
        app:layout_row="0"
        app:layout_column="1"
        app:layout_columnWeight="1"
        android:layout_width="0dp"
        android:layout_height="50dp" />
    
    <!-- Ячейки данных -->
    <TextView
        android:text="1-1"
        android:gravity="center"
        android:padding="8dp"
        app:layout_row="1"
        app:layout_column="0"
        app:layout_columnWeight="1"
        android:layout_width="0dp"
        android:layout_height="50dp" />
    
    <!-- ... дополнительные ячейки -->
</GridLayout>
```

#### Java код (ProgrammaticGridLayoutActivity.java)
```java
GridLayout grid = new GridLayout(this);
grid.setRowCount(3);
grid.setColumnCount(3);

// Заголовок
for (int col = 0; col < 3; col++) {
    TextView header = new TextView(this);
    header.setText("Столбец " + (col + 1));
    header.setBackgroundColor(0xFF1976D2);
    header.setTextColor(Color.WHITE);
    header.setGravity(Gravity.CENTER);
    
    GridLayout.LayoutParams params = new GridLayout.LayoutParams();
    params.rowSpec = GridLayout.spec(0);
    params.columnSpec = GridLayout.spec(col, 1f); // 1f = вес
    params.width = 0;  // ширина вычисляется из веса
    params.height = 50;
    
    header.setLayoutParams(params);
    grid.addView(header);
}

// Ячейки данных
for (int row = 1; row < 3; row++) {
    for (int col = 0; col < 3; col++) {
        TextView cell = new TextView(this);
        cell.setText(row + "-" + col);
        cell.setGravity(Gravity.CENTER);
        
        GridLayout.LayoutParams params = new GridLayout.LayoutParams();
        params.rowSpec = GridLayout.spec(row);
        params.columnSpec = GridLayout.spec(col, 1f);
        params.width = 0;
        params.height = 50;
        
        cell.setLayoutParams(params);
        grid.addView(cell);
    }
}
```

### Пример: Объединение ячеек (Span)
```xml
<TextView
    android:text="Заголовок (объединено)"
    app:layout_row="0"
    app:layout_column="0"
    app:layout_columnSpan="3"  <!-- объединяет 3 столбца -->
    app:layout_rowSpan="2"     <!-- занимает 2 строки -->
    android:layout_width="0dp" />
```

## 6. ScrollView / HorizontalScrollView: Прокручиваемое содержимое

### Описание
- **ScrollView** - вертикальная прокрутка (содержит ONE LinearLayout с фиксированной шириной)
- **HorizontalScrollView** - горизонтальная прокрутка (содержит ONE LinearLayout с фиксированной высотой)

### Правила использования
1. ScrollView может содержать **ТОЛЬКО ОДИН** прямой дочерний элемент (обычно LinearLayout)
2. Этот дочерний элемент должен иметь `layout_width="match_parent"` и `layout_height="wrap_content"`

### Пример: Вертикальная прокрутка

#### XML (layout_scroll.xml)
```xml
<ScrollView
    android:layout_width="match_parent"
    android:layout_height="150dp"
    android:background="@android:color/white">
    
    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:orientation="vertical">
        
        <TextView
            android:layout_width="match_parent"
            android:layout_height="wrap_content"
            android:text="Элемент 1" />
        
        <TextView
            android:layout_width="match_parent"
            android:layout_height="wrap_content"
            android:text="Элемент 2" />
        
        <!-- ... больше элементов для прокрутки -->
    </LinearLayout>
</ScrollView>
```

#### Java код (ProgrammaticScrollViewActivity.java)
```java
ScrollView scrollView = new ScrollView(this);
scrollView.setLayoutParams(new LinearLayout.LayoutParams(
    LinearLayout.LayoutParams.MATCH_PARENT, 150
));

LinearLayout contentLayout = new LinearLayout(this);
contentLayout.setOrientation(LinearLayout.VERTICAL);
contentLayout.setLayoutParams(new ViewGroup.LayoutParams(
    ViewGroup.LayoutParams.MATCH_PARENT,
    ViewGroup.LayoutParams.WRAP_CONTENT
));

// Добавляем элементы для прокрутки
for (int i = 0; i < 10; i++) {
    TextView item = new TextView(this);
    item.setText("Элемент #" + (i + 1));
    item.setPadding(16, 8, 16, 8);
    item.setBackgroundColor(i % 2 == 0 ? Color.WHITE : 0xFFF5F5F5);
    contentLayout.addView(item);
}

scrollView.addView(contentLayout);
```

### Пример: Горизонтальная прокрутка
```xml
<HorizontalScrollView
    android:layout_width="match_parent"
    android:layout_height="120dp">
    
    <LinearLayout
        android:layout_width="wrap_content"
        android:layout_height="match_parent"
        android:orientation="horizontal">
        
        <!-- Элементы располагаются горизонтально -->
        <TextView android:layout_width="100dp" android:layout_height="match_parent"
            android:text="Карточка 1" />
        <TextView android:layout_width="100dp" android:layout_height="match_parent"
            android:text="Карточка 2" />
    </LinearLayout>
</HorizontalScrollView>
```

## 7. Gravity vs Layout_Gravity

### gravity - выравнивание СОДЕРЖИМОГО внутри элемента
```xml
<TextView
    android:gravity="center"  <!-- выравнивает текст внутри TextView -->
    android:text="Центр" />

<LinearLayout
    android:gravity="center"  <!-- выравнивает детей - центр их -->
    android:orientation="vertical">
</LinearLayout>
```

### layout_gravity - выравнивание ЭЛЕМЕНТА внутри контейнера
```xml
<LinearLayout android:orientation="horizontal">
    <TextView
        android:layout_gravity="top"          <!-- элемент приклеится к верху -->
        android:text="Вверху" />
    
    <TextView
        android:layout_gravity="center_vertical"  <!-- элемент отцентрируется вертикально -->
        android:text="Центр" />
    
    <TextView
        android:layout_gravity="bottom"       <!-- элемент приклеится к низу -->
        android:text="Внизу" />
</LinearLayout>
```

## 8. Вложенные Layouts (Nested Layouts)

### Описание
Контейнеры можно вкладывать друг в друга для построения сложных интерфейсов.

### Пример: LinearLayout содержит TableLayout + GridLayout

```xml
<LinearLayout android:orientation="vertical">
    <!-- Верхняя часть: TableLayout -->
    <TableLayout android:layout_height="100dp">
        <!-- таблица -->
    </TableLayout>
    
    <!-- Разделитель -->
    <View android:layout_height="1dp" android:background="#CCCCCC" />
    
    <!-- Нижняя часть: GridLayout -->
    <GridLayout android:columnCount="3">
        <!-- сетка -->
    </GridLayout>
</LinearLayout>
```

### Include элемент (переиспользование layouts)

```xml
<!-- component_card.xml - переиспользуемый компонент -->
<LinearLayout>
    <TextView android:id="@+id/title" android:text="Заголовок" />
    <TextView android:id="@+id/description" android:text="Описание" />
</LinearLayout>

<!-- layout_main.xml - главный layout -->
<LinearLayout android:orientation="vertical">
    <include
        layout="@layout/component_card"
        android:layout_width="match_parent"
        android:layout_height="wrap_content" />
    
    <include
        layout="@layout/component_card"
        android:layout_width="match_parent"
        android:layout_height="wrap_content" />
</LinearLayout>
```

## 9. Практические советы

### ✅ Лучшие практики
1. **Используйте ConstraintLayout** для сложных интерфейсов (более гибкий, чем RelativeLayout)
2. **Избегайте глубокой вложенности** (максимум 5-7 уровней)
3. **Используйте weight в LinearLayout** вместо подсчета размеров вручную
4. **Кэшируйте DisplayMetrics.density** для преобразования dp ↔ px
5. **Предпочитайте XML** для статичных layouts (более читаемо, легче интернационализировать)
6. **Используйте Java код** когда layout динамически генерируется

### 🚫 Чего избегать
- Вложение многих RelativeLayout один в другой (сложность расчета)
- ScrollView в ScrollView (проблемы с прокруткой)
- Слишком больших layouts (более 50+ элементов) без оптимизации
- Hardcoded размеры вместо dp-значений

```java
new LinearLayout.LayoutParams(
    int width,          // MATCH_PARENT, WRAP_CONTENT, или dp
    int height,         // MATCH_PARENT, WRAP_CONTENT, или dp
```

#### RelativeLayout.LayoutParams
```java
new RelativeLayout.LayoutParams(int width, int height);
```

```java

    int width, int height,
    int gravity  // Gravity.TOP, CENTER, BOTTOM и комбинации

```

#### GridLayout.LayoutParams
```java
GridLayout.LayoutParams params = new GridLayout.LayoutParams();

params.columnSpec = GridLayout.spec(int column, float weight);
params.width = 0;   // когда используется weight
params.height = 0;  // когда используется weight
```
