## 🎯 Основные достижения

### 1. LinearLayout - Всё что нужно знать
- ✓ Равномерное распределение (weight 1,1,1)
- ✓ Неравномерное распределение (weight 1,2,3)
- ✓ layout_gravity (top/center/bottom)
- ✓ Вертикальная ориентация
- ✓ Горизонтальная ориентация

**Файлы:**
- XML: `layout_linear_weight.xml` (5 примеров)
- Java: `ProgrammaticLinearLayoutActivity.java` (5 примеров в коде)

### 2. RelativeLayout - Позиционирование
- ✓ Выравнивание по границам родителя (alignParent*)
- ✓ Позиционирование относительно других элементов (rightOf, below, above)
- ✓ Выравнивание границ элементов (align*)
- ✓ Центрирование (centerHorizontal, centerVertical, centerInParent)

**Файлы:**
- XML: `layout_relative.xml` (4 примера)
- Java: `ProgrammaticRelativeLayoutActivity.java` (5 позиционирований)

### 3. TableLayout - Табличные данные
- ✓ Базовая таблица 3x3
- ✓ Объединение ячеек (columnSpan)
- ✓ Стилизованные строки (alternating colors)
- ✓ Растяжение столбцов (stretchColumns)

**Файлы:**
- XML: `layout_table.xml` (3 примера)
- Java: `ProgrammaticTableLayoutActivity.java` (3 варианта таблиц)

### 4. FrameLayout - Слоистые интерфейсы
- ✓ Фоновый слой + несколько перекрывающихся элементов
- ✓ Z-order (порядок добавления влияет на видимость)
- ✓ Все 9 позиций gravity (top|start, center, bottom|end и т.д.)
- ✓ Прозрачность (alpha в цветах)
- ✓ Паттерн "badge" (значок сверху)

**Файлы:**
- XML: `layout_frame.xml` (4 примера: базовый, 9 позиций, Z-order, badge)
- Java: `ProgrammaticFrameLayoutActivity.java` (3 примера)

### 5. GridLayout - Гибкие сетки
- ✓ Базовая сетка 3x3 с равномерной шириной
- ✓ Объединение ячеек (rowSpan, columnSpan)
- ✓ Распределение ширины (layout_columnWeight)
- ✓ Span комбинации (2x2 слияния, полные строки)

**Файлы:**
- XML: `layout_grid.xml` (3 примера)
- Java: `ProgrammaticGridLayoutActivity.java` (3 варианта)

### 6. ScrollView - Прокручиваемое содержимое
- ✓ Вертикальная прокрутка (ScrollView)
- ✓ Горизонтальная прокрутка (HorizontalScrollView)
- ✓ Комбинированная прокрутка (ScrollView + GridLayout)
- ✓ Таблица в ScrollView

**Файлы:**
- XML: `layout_scroll.xml` (3 примера)
- Java: `ProgrammaticScrollViewActivity.java` (3 примера)

### 7. Gravity vs Layout_Gravity
- ✓ gravity - выравнивание СОДЕРЖИМОГО внутри элемента
- ✓ layout_gravity - выравнивание ЭЛЕМЕНТА внутри контейнера
- ✓ 4 подробных примера с визуализацией

**Файлы:**
- XML: `layout_gravity.xml` (4 примера)

### 8. Вложенные Layouts (Nested)
- ✓ LinearLayout → FrameLayout → LinearLayout → GridLayout (4 уровня)
- ✓ Include элемент (component_card.xml используется 2 раза)
- ✓ Таблица-подобная структура с labels и values
- ✓ Карточка-подобная верстка

**Файлы:**
- XML: `layout_nested.xml` (4 примера вложенности)
- XML: `component_card.xml` (переиспользуемая карточка)

## 📚 Обучающие примеры

Каждый контейнер демонстрирует:
1. **"Hello World" версия** - самое простое использование
2. **Реальный примере** - таблица, карточки, сетка данных
3. **Advanced версия** - комбинирование, стилизация, animation-ready

### Ключевые паттерны

**Pattern 1: Programmatic создание с LayoutParams**
```java
LinearLayout.LayoutParams params = new LinearLayout.LayoutParams(width, height);
params.setMargins(left, top, right, bottom);
params.gravity = Gravity.CENTER; // if applicable
view.setLayoutParams(params);
```

**Pattern 2: Density-aware sizing**
```java
float density = getResources().getDisplayMetrics().density;
int pixels = (int)(dpValue * density);
```

**Pattern 3: Color with alpha**
```java
// 0xAARRGGBB format
int transparentBlue = 0xCC2196F3;  // 80% opacity
int opaqueRed = 0xFFD32F2F;        // 100% opacity
```

|-----------|-------------|----------|
| LinearLayout | Both | Simple linear UI, weight distribution |
| RelativeLayout | N/A | Relative positioning, complex overlaps |
| TableLayout | Vertical | Tabular data, fixed columns |
| FrameLayout | N/A | Overlays, badges, Z-ordering |
| GridLayout | Both | Data grids, dynamic columns |
| ScrollView | Vertical | Long content lists |
